## 📁 仓库结构

```
prompt-engineering-repo/
│
├── prompts/                  # 所有提示词，按维度分类
│   ├── by-tool/              # 按 AI 工具分类
│   ├── by-task/              # 按任务类型分类
│   └── by-industry/          # 按行业场景分类
│
├── templates/                # 可复用的提示词骨架模板
├── frameworks/               # 提示词设计方法论（CO-STAR / RISEN 等）
├── evals/                    # 提示词效果测试与评估记录
├── examples/                 # 完整的端到端使用案例
├── docs/                     # 文档（贡献指南、命名规范等）
├── scripts/                  # 辅助脚本（批量测试、格式转换等）
└── assets/                   # 截图、效果图等媒体资源
```

---

## 🗂️ 分类说明

### 按工具 (`by-tool/`)
适合**已知要用哪个工具**的情况，直接找对应工具的最佳提示词。

### 按任务 (`by-task/`)
适合**已知要做什么任务**的情况，跨工具通用。

### 按行业 (`by-industry/`)
适合**垂直场景需求**，包含行业专有术语和上下文。
